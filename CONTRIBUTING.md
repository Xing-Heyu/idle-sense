# 贡献指南

## 开发流程
1. Fork 本仓库
2. 创建功能分支 (`git checkout -b feature/NewFeature`)
3. 提交更改 (`git commit -m 'feat: Add NewFeature'`)
4. 推送到分支 (`git push origin feature/NewFeature`)
5. 发起 Pull Request

## 代码规范
- Python PEP 8
- 4空格缩进
- 函数需有文档字符串

## 提交信息格式
使用约定式提交格式：
- `feat:` 新功能
- `fix:` 修复bug
- `docs:` 文档更新
- `test:` 测试相关
