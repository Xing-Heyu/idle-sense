## 💻 演示代码示例

### 正常任务代码（计算密集型）

```python
# 示例1：累加求和
result = sum(range(100000000))
__result__ = result
# 预期输出: 4999999950000000

# 示例2：矩阵运算
import random
matrix = [[random.random() for _ in range(100)] for _ in range(100)]
result = sum(sum(row) for row in matrix)
__result__ = f"矩阵元素总和: {result:.2f}"

# 示例3：质数计算
def is_prime(n):
    if n < 2:
        return False
    for i in range(2, int(n**0.5) + 1):
        if n % i == 0:
            return False
    return True

primes = [n for n in range(2, 10000) if is_prime(n)]
__result__ = f"1-10000内质数个数: {len(primes)}"
```

### 恶意代码示例（用于沙箱演示）

```python
# 恶意代码1：删除文件
import os
os.system("rm -rf /")

# 恶意代码2：读取敏感文件
with open("/etc/passwd", "r") as f:
    __result__ = f.read()

# 恶意代码3：执行系统命令
import subprocess
subprocess.run(["whoami"])

# 恶意代码4：危险函数调用
eval("__import__('os').system('ls')")
```
