# 🎯 闲置计算加速器 - 超级小白保姆级使用指南

> **写给完全不懂电脑的小白**：本指南假设您连复制粘贴都不太会，一步一步教您使用这个软件

---

## 📋 开始前必读（非常重要！）

### 🤔 这个软件是做什么的？
简单说，就是让您的电脑在"没事干"的时候帮您做计算工作，就像雇佣一个免费的"计算工人"。

### 💡 您需要准备什么？
1. 一台Windows电脑（Win10或Win11）
2. 能上网
3. 有耐心按照步骤操作（最重要！）

---

## 🚀 第一步：下载和安装软件（5分钟）

### 1.1 下载软件
1. 找到软件下载链接（从给您软件的人那里获取）
2. 点击下载，选择"保存到桌面"
3. 等待下载完成（通常1-2分钟）

### 1.2 解压软件（如果下载的是压缩包）
1. 在桌面上找到下载的文件（通常叫`idle-sense.zip`或类似名字）
2. 右键点击文件，选择"全部解压缩..."
3. 点击"提取"按钮
4. 等待解压完成，桌面上会出现一个叫`idle-sense`的文件夹

### 1.3 检查文件夹内容
打开`idle-sense`文件夹，您应该看到很多文件，其中最重要的是：
- `start_all.bat`（一键启动按钮）
- `web_interface.py`（网页界面）
- 其他文件（不用管它们）

---

## 🎮 第二步：启动软件（3分钟）

### 2.1 一键启动（最简单的方法）
1. 双击`start_all.bat`文件
2. 会弹出黑色窗口，**不要关闭这些窗口！**
3. 等待1-2分钟，直到看到类似"网页界面已启动"的提示

### 2.2 打开网页界面
1. 打开您的浏览器（推荐用Chrome或Edge）
2. 在地址栏输入：`http://localhost:8501`
3. 按回车键
4. 您应该看到一个漂亮的网页界面，标题是"闲置计算加速器"

### 2.3 检查是否成功
如果您能看到网页界面，恭喜您！软件已经成功启动了！

---

## 👤 第三步：注册账号（2分钟）

### 3.1 找到注册页面
1. 在网页界面左侧，找到"用户管理"部分
2. 点击"注册"按钮

### 3.2 填写注册信息
1. **用户名**：输入您喜欢的名字（比如"小明"或"张三"）
2. **邮箱**：输入您的邮箱地址（比如`xiaoming@email.com`）
3. **文件夹位置**：选择"项目目录"（最简单，推荐）

### 3.3 同意协议（必须做！）
1. 找到"用户协议与权限确认"部分
2. **必须勾选**"✅ 我已阅读并同意用户协议，并确认系统权限获取"
3. 点击"注册"按钮

### 3.4 完成注册
1. 等待几秒钟
2. 看到"注册成功"提示
3. 点击"登录"标签页
4. 输入您刚才注册的用户名和密码
5. 点击"登录"

---

## 📁 第四步：准备您的数据文件（5分钟）

### 4.1 找到您的专属文件夹
1. 注册成功后，系统会在您的电脑上创建一个专属文件夹
2. 位置通常是：`C:\idle-sense\node_data\user_data\您的用户ID`
3. 您可以通过以下方式找到它：
   - 打开"此电脑"
   - 在地址栏输入：`C:\idle-sense\node_data\user_data`
   - 找到以您的用户名命名的文件夹

### 4.2 准备数据文件（示例）
假设您有一些数字需要计算，比如：
```
10,20,30,40,50
```

1. 打开"记事本"程序
2. 输入上面的数字
3. 点击"文件" → "另存为"
4. 在"保存类型"中选择"所有文件"
5. 文件名输入：`my_numbers.txt`
6. 保存到您的专属文件夹中

### 4.3 检查文件是否正确
1. 打开您的专属文件夹
2. 确认`my_numbers.txt`文件在里面
3. 双击打开，确认内容正确

---

## 🧮 第五步：创建您的第一个计算任务（5分钟）

### 5.1 进入任务提交页面
1. 在网页界面顶部，点击"提交任务"标签页
2. 选择"单节点任务"（先从简单的开始）

### 5.2 输入计算代码
1. 在"输入Python代码"框中，输入以下代码（**完全复制粘贴即可**）：

```python
# 读取您的数据文件
try:
    # 读取您刚才保存的文件
    data_content = read_user_file("my_numbers.txt")
    print("成功读取文件内容！")
    
    # 处理数据
    numbers = data_content.strip().split(',')
    total = 0
    for num in numbers:
        total += int(num.strip())
    
    print(f"所有数字的总和是: {total}")
    print(f"平均值是: {total / len(numbers)}")
    
except Exception as e:
    print(f"出错了: {e}")
    print("请确认文件名是否正确，文件是否在用户文件夹中")
```

### 5.3 设置任务参数
1. **超时时间**：保持默认的300秒（5分钟）
2. **CPU需求**：保持默认的1.0
3. **内存需求**：保持默认的512MB

### 5.4 提交任务
1. 点击"🚀 提交单节点任务"按钮
2. 等待几秒钟
3. 看到"任务提交成功"的提示

---

## 📊 第六步：查看计算结果（2分钟）

### 6.1 进入任务监控页面
1. 在网页界面顶部，点击"任务监控"标签页
2. 您应该看到您刚才提交的任务

### 6.2 查看任务状态
1. 任务状态会从"待处理" → "运行中" → "已完成"
2. 等待任务状态变为"已完成"（通常几秒钟）

### 6.3 查看计算结果
1. 点击任务旁边的"查看详情"
2. 展开"执行结果"
3. 您应该看到类似这样的输出：
```
成功读取文件内容！
所有数字的总和是: 150
平均值是: 30.0
```

## 🔧 常见问题解决（小白专用）

### 问题1：双击start_all.bat没反应
**解决方法**：
1. 右键点击`start_all.bat`
2. 选择"以管理员身份运行"
3. 如果弹出安全提示，点击"是"

### 问题2：网页打不开（localhost:8501）
**解决方法**：
1. 确认黑色窗口没有关闭
2. 等待1-2分钟再刷新网页
3. 检查地址是否输入正确（不要有空格）

### 问题3：注册时提示"文件夹创建失败"
**解决方法**：
1. 选择"项目目录"而不是C盘或D盘
2. 确保以管理员身份运行
3. 关闭杀毒软件重试

### 问题4：任务提交失败
**解决方法**：
1. 检查代码是否完全复制粘贴（不要有多余的空格）
2. 确认文件名是否正确（`my_numbers.txt`）
3. 确认文件在正确的文件夹中

### 问题5：看不到计算结果
**解决方法**：
1. 在任务监控页面点击"刷新任务列表"
2. 等待任务状态变为"已完成"
3. 点击任务旁边的"查看详情"

## 💡 进阶使用（可选）

如果您有更多数据需要处理：

如果您有多台电脑：

2. 选择"分布式任务"
3. 系统会自动分配任务给所有电脑
